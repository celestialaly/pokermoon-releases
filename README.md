# Poker Moon

A desktop application for analyzing your Winamax poker hand histories.

Import your tournament and hand history files, browse your statistics, and track your winnings over time — entirely offline, no account required.

## Features

- **Automatic import** — points Poker Moon at your Winamax history folder; it discovers and imports all tournament summaries and hand histories
- **Dashboard** — KPI cards (hands played, tournaments played) and a cumulative winnings chart
- **Tournaments** — sortable, searchable table of all your tournaments with buy-in cost, final position and net gain
- **Hands** — browse individual hands with card display, filterable by tournament
- **Multi-account** — configure multiple Winamax usernames and switch between them
- **Auto-update** — the app checks for new releases at startup and lets you install them in one click (stable and pre-release channels)

## Bug report / Request a new feature or poker site

Bug reports and feature requests are welcome. Please open issues on the current **[pokermoon-releases](https://github.com/celestialaly/pokermoon-releases/issues)** repository.

When reporting a bug, include:
- Your OS and architecture (e.g. Windows x86_64, macOS Apple Silicon)
- The version of Poker Moon (`Help → About` or the footer of the app)
- Steps to reproduce and, if relevant, the hand history file that triggers the issue (you can anonymize player names)

For feature requests, describe the use case rather than the implementation — it helps prioritise.
